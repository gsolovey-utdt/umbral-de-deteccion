Que mide este experimento

Esta tarea estima cuanta diferencia de color necesitas para localizar el borde entre dos regiones contiguas.

El resultado final es un umbral individual. Un valor mas bajo indica mayor sensibilidad para detectar cambios sutiles en ese canal de color.

El umbral se estima a partir de la proporcion de aciertos por nivel de diferencia y un criterio de rendimiento de 0.5.
